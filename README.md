My Weather
Welcome to My Weather, a web application that provides accurate and up-to-date weather information for any location. Developed by Shreshth Gupta, this application integrates various weather APIs to deliver detailed weather data in an attractive and user-friendly interface.

Features
Current Weather: Get real-time weather conditions including temperature, humidity, wind speed, and more.
Weather Forecast: Access detailed forecasts for upcoming days to plan your activities.
Interactive Map: View weather conditions on an interactive map with various layers for different weather parameters.
Attractive Design: Enjoy a visually appealing interface with customizable themes and backgrounds.
Responsive Layout: Access the application on any device with a responsive and adaptive design.

Technologies Used
Frontend: HTML, CSS, JavaScript, EJS (Embedded JavaScript templating)
Backend: Node.js, Express.js
APIs: OpenWeatherMap, IPInfo
Libraries: Leaflet (for interactive maps)
Installation
Clone the repository:

bash:
git clone https://github.com/Shreshth-gupta/My-Weather.git
cd My-Weather
Install the dependencies:

bash:
npm install
Create a .env file in the root directory and add your API keys:

env:
OPENWEATHERMAP_API_KEY=your_openweathermap_api_key
IPINFO_API_KEY=your_ipinfo_api_key
Start the application:

bash:
npm start
Open your browser and go to http://localhost:3000 to view the application.

Usage
Search for Weather: Enter a city name or use the interactive map to get weather details for any location.
View Forecast: Check the weather forecast for the next few days.
Customize Appearance: Change themes and backgrounds to suit your preferences.
